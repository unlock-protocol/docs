          In the Unlock ecosystem...
          <dl>
            <dt>Protocol</dt>
              <dd>A protocol is an open-source collection of code that typically 
              solves a specific problem (e.g. transferring information online or, 
              in Unlock’s case, creating and managing memberships and subscriptions). 
              Most notably, open-source protocols can be used by anyone without 
              the need to ask for permission and can be combined with other 
              protocols (in the same manner as LEGO bricks) to create more 
              sophisticated systems.</dd>
            <dt>Lock</dt>
            <dd>a “Lock” is a smart contract that creates 
            (or “mints”) NFTs. Locks also can be used to ensure “members-only” 
            access to resources like online content, live events, online 
            certifications, or other “members-only” experiences.</dd>
            <dt>Key</dt>
            <dd>a “Key” is an NFT (minted by a Lock) 
            that has additional features added to it to facilitate the management 
            of memberships, such as an expiration date.</dd>
          </dl>
